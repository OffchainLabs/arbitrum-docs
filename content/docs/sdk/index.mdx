# Arbitrum SDK

The Arbitrum SDK is a powerful TypeScript library that streamlines interactions with Arbitrum networks. It offers robust tools for bridging tokens and passing messages between networks through an intuitive interface to the underlying smart contracts.

## Key Features

- **Token Bridging**: Effortlessly bridge tokens between Ethereum and Arbitrum networks
- **Message Passing**: Seamlessly pass messages across networks with built-in retry mechanisms
- **Contracts Interface**: Leverage a strongly-typed interface for interacting with smart contracts
- **Network Support**: Built-in support for Arbitrum One, Nova, and custom Orbit chains

### Bridge ETH to Arbitrum One

```ts
import { getArbitrumNetwork, EthBridger } from '@arbitrum/sdk'
import { utils } from 'ethers'

// Get the Arbitrum network configuration
const childNetwork = await getArbitrumNetwork(42161) // Arbitrum One
const ethBridger = new EthBridger(childNetwork)

// Deposit ETH from Ethereum to Arbitrum One
const ethDepositTxResponse = await ethBridger.deposit({
  amount: utils.parseEther('0.1'),
  parentSigner, // ethers v5 signer connected to Ethereum mainnet
  childProvider, // ethers v5 provider connected to Arbitrum One
})

const ethDepositTxReceipt = await ethDepositTxResponse.wait()
console.log('ETH deposited successfully!')
```

### Withdraw ETH from Arbitrum One

```ts
// Withdraw ETH from Arbitrum One to Ethereum
const withdrawTx = await ethBridger.withdraw({
  amount: utils.parseEther('0.1'),
  childSigner, // ethers v5 signer connected to Arbitrum One
  destinationAddress: await childSigner.getAddress(),
})

const withdrawReceipt = await withdrawTx.wait()
console.log('ETH withdrawal initiated!')
```

</Steps>

## Core Concepts

### Asset Bridgers

The SDK provides specialized bridgers for different asset types:

- **[EthBridger](./assetBridger/ethBridger)** - For bridging ETH between networks
- **[Erc20Bridger](./assetBridger/erc20Bridger)** - For bridging ERC-20 tokens
- **[L1L3Bridger](./assetBridger/l1l3Bridger)** - For direct L1 to L3 bridging

### Cross-Chain Messages

When assets are moved, cross-chain messages are automatically created. The SDK provides classes to track and manage these messages:

- **[ParentToChildMessage](./message/ParentToChildMessage)** - Track deposits and L1→L2 messages
- **[ChildToParentMessage](./message/ChildToParentMessage)** - Track withdrawals and L2→L1 messages

### Network Configuration

The SDK comes pre-configured for Arbitrum networks, but can be extended for custom Orbit chains:

```ts
import { registerCustomArbitrumNetwork } from '@arbitrum/sdk'

registerCustomArbitrumNetwork({
  chainId: 123456,
  name: 'My Custom Orbit Chain',
  parentChainId: 1, // Ethereum mainnet
  // ... other configuration
})
```
