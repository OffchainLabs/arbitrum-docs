The onchain configuration object that defines a [Data Availability Committee (DAC)](/intro/glossary#data-availability-committee-dac)—the [BLS public keys](/intro/glossary#bls-signature) of every committee member and the signature threshold required to produce a valid [Data Availability Certificate](/intro/glossary#data-availability-certificate). A new keyset is published whenever committee membership or the threshold changes.
