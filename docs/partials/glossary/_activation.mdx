The step that prepares a deployed [Stylus](/intro/glossary#stylus) contract to be called. Activation registers the program with [ArbOS](/intro/glossary#arbos), validates the [WASM](/intro/glossary#wasm) bytecode, and primes the per-node compiled-artifact cache so subsequent calls can execute the program at native speed. Activation costs gas proportional to program size, and re-activation is required after the cache entry expires (365 days by default) or after an ArbOS upgrade that changes the [WASM module root](/intro/glossary#wasm-module-root).
