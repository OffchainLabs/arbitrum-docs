A [Validator](/intro/glossary#validator) configured to actively submit new [Assertions](/intro/glossary#assertion) to the parent chain under [BoLD](/intro/glossary#bold). Proposers post the largest bonds in the staking hierarchy to deter delay attacks. Contrast with a [Watchtower Validator](/intro/glossary#watchtower-validator), which only observes, and a [Defensive Validator](/intro/glossary#defensive-validator), which acts only to counter an invalid claim.
