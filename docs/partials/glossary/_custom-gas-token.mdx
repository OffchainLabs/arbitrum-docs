---
title: Custom gas token
key: custom-gas-token
titleforSort: Custom gas token
---

A non-ETH ERC-20 token chosen at [Arbitrum chain](/intro/glossary#arbitrum-chain) deployment time to be used for paying transaction fees on that chain. The custom gas token replaces ETH everywhere ETH would otherwise be charged for gas, and is bridged from the [parent chain](/intro/glossary#parent-chain) like any other token.
