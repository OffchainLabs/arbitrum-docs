### Installing cargo-stylus

The Stylus CLI is your toolkit for speeding up the development process. It handles project scaffolding, building contracts to WASM, checking onchain activation, and deploying your contracts.

<VanillaAdmonition type="warning" title="Windows users need WSL">

`cargo-stylus` requires a Unix environment. On Windows, install [WSL 2](https://learn.microsoft.com/en-us/windows/wsl/install) and run all `cargo stylus` commands from the WSL terminal — native Windows command prompts are not supported.

</VanillaAdmonition>

The `cargo-stylus` CLI tool helps you create, build, and deploy Stylus contracts.

**Install:**

```shell
cargo install cargo-stylus
```

**Verify:**

```shell
cargo stylus --version
```

**Update:**

```shell
cargo install cargo-stylus --force
```
